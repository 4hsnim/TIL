2. 1주차 주요 개념 키워드 적어보기
 - Git, Github, git initialize, staging(add), repo, commit, sourcetree, clone, push, pull, 버전관리
  . Git : 협업하기 위해 코딩 파일뿐만 아니라 워드파일같은 것을 공유하고, 관리하는 것
  . Github : 구글드라이브같이 협업파일 클라우드서비스 제공
  . Git initialize : PC안에 있는 프로젝트를 Git으로 프로젝트를 끌고 오는 행위
  . staging(add) : commit을 하기 위해 파일을 올리는 행위
  . repo : repsitory의 약자, Git으로 관리되는 포토젝트
  . commit : 현재 프로젝트를 하고 있는 순간을 저장하는 행위
  . sourcetree : Git을 효율적으로 관리하게 해주는 프로그램
  . push : 로컬 repo → 원격 repo로 commit을 반영해주는 것
  . pull  : 원격 repo → 로컬 repo로  commit을 반영해주는 것
  . clone : 로컬 repo에 파일이 아무것도 없을 때 원격 repo에서 쓴 파일을 그대로 가져오는 것.
  . 버전관리 : 프로젝트 상태가 변경되는 것을 관리하는 것.
